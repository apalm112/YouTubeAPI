About APIs

An Application Programming Interface (API) is an interface that provides developers access to a application's functionality so that a developer can write customized applications to it. An example of an API is a web browser's Document Object Model (DOM) which is the web browser's API that allows scripts to access and change its contents. There are many other types of APIs published for just about any software "product" or "platform", for example Apple OS X, Microsoft Office, Google Maps, or even Facebook, all of which allows a developer to create interesting projects that interface with what these APIs have to offer.
